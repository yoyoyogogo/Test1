🧩 Pegar Modelo de IA
Pídele a una IA (ChatGPT, Gemini, Claude…):
"Crea una [cama / árbol / mesa / …] en Three.js usando MeshPhongMaterial.
El código debe ser el cuerpo de una función que recibe THREE como parámetro
y hace return de un THREE.Group con toda la geometría."

💡 Ejemplo real — código de la Silla:
const mat = new THREE.MeshPhongMaterial({ color: 0x8B6914 });
const legCol = new THREE.Color(0x8B6914).multiplyScalar(0.7);
const legMat = new THREE.MeshPhongMaterial({ color: legCol });
const group = new THREE.Group();
// Asiento
const seat = new THREE.Mesh(new THREE.BoxGeometry(1.2, 0.1, 1.2), mat);
seat.position.y = 0.6; group.add(seat);
// Respaldo
const back = new THREE.Mesh(new THREE.BoxGeometry(1.2, 1.0, 0.15), mat);
back.position.set(0, 1.1, -0.525); group.add(back);
// 4 Patas
const legGeo = new THREE.BoxGeometry(0.15, 0.6, 0.15);
[[-0.5,0.3,-0.5],[0.5,0.3,-0.5],[-0.5,0.3,0.5],[0.5,0.3,0.5]].forEach(p => {
  const leg = new THREE.Mesh(legGeo, legMat);
  leg.position.set(p[0], p[1], p[2]); group.add(leg);
});
return group;
